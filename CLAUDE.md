# Meow Coffee — Кофейня

## Запуск
Статический сайт — открыть `index.html` в браузере или через http-server.

## Стек
- HTML + CSS + vanilla JS (без фреймворков)
- Иконки: Feather Icons (CDN)
- Шрифты: Google Fonts (Playfair Display, Inter)
## Структура
```
index.html    — единственная страница (лендинг)
script.js     — вся логика: переводы, меню, формы, таймер, анимации
styles.css    — все стили: тёмная тема, золотые акценты
img/          — изображения (латинские имена файлов)
```

## Дизайн
- Тёмная тема: фон #0d0d0d, золотые акценты #c8a97e, текст #f5f0e8
- Glass-panel эффект
- Кастомный курсор (точка + кольцо)

## Функционал
1. Навигация — якорные ссылки, мобильное меню, подсветка активного пункта
2. Hero — параллакс при скролле
3. Таймер счастливых часов (17:00–19:00, скидка 20%)
4. О нас — анимированные счётчики статистики
5. Процесс — 4 этапа приготовления кофе
6. Меню — 13 позиций с фильтрацией (кофе, авторские, десерты)
7. Команда — 3 сотрудника
8. Галерея — 6 фото, lightbox
9. Отзывы — карусель с авто-прокруткой
10. Бронирование — форма с валидацией
11. Контакты — адрес, часы, Google Maps
12. Прелоадер — пульс логотипа + полоса загрузки

## Мультиязычность (RU/EN)
- Атрибуты `data-i18n` + словарь в `script.js`
- Язык сохраняется в localStorage

## Анимации
- Прелоадер, параллакс hero, scroll fade-in, кастомный курсор, lightbox, карусель, счётчики

## Правила
- Язык общения: русский
- Не добавлять фреймворки
